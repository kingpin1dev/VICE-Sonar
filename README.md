## Features

### 🎧 Audio Controls
- Volume sliders and mute toggles for all Sonar channels: **Master, Game, Chat, Media, Aux, Mic**
- **ChatMix** — hardware-synced Game/Chat balance slider

### 🎛 EQ Profiles
- Switch between your favorite EQ presets per device
- Supports **Game, Chat, Mic, Media, Aux** channels independently

### 💡 RGB Lighting
- Control SteelSeries device colors via **GameSense**
- Quick presets + custom RGB picker

### ⚙️ Settings
- **7 languages** — Turkish, English, French, Russian, German, Spanish, Arabic
- **Theme switcher** — multiple themes with horizontal layout option
- **Transparency slider** — adjust window opacity
- **Always on top** toggle
- **Windows startup** option

### ✨ UI & UX
- Borderless & draggable — sits anywhere on screen
- Position memory between sessions
- Instant tab switching
- System tray support
- No taskbar entry
